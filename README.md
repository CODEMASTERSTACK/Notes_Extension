## ğŸ› ï¸ Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/CODEMASTERSTACK/Notes_Extension.git
    ```
2.  **Open Chrome Extensions**
    - Navigate to `chrome://extensions/` in your browser.
3.  **Enable Developer Mode**
    - Toggle the switch in the top-right corner.
4.  **Load Unpacked**
    - Click the "Load unpacked" button.
    - Select the folder wherever you cloned/downloaded this repository.
5.  **Pin & use!**
    - Pin the extension icon to your toolbar for easy access.

## ğŸ“– Usage

1.  **Open the Panel**: Click the extension icon to open the Side Panel.
2.  **Create a Note**: Click the **+** button.
3.  **Clip Content**: Highlight text on a page -> Right-click -> **Save to Notes**.
4.  **Edit**: Click on any note to open the rich editor.
5.  **Export**: Use the save dropdown in the editor toolbar to download your note.
